Paleta de Cores

As cores são a sessão fundamental do design system, elas que definem a identidade visual do MyBuddy, e serão usadas em todas as partes da nossa interface, desde simples botões até componentes mais complexos, como cards e componentes de navegação. A paleta de cores do MyBuddy é composta por uma combinação de cores primárias, secundárias, de destaque, de texto e de fundo, que foram cuidadosamente selecionadas para criar uma identidade visual forte e consistente. As cores são usadas para criar contraste, hierarquia visual e para transmitir emoções e mensagens específicas, ajudando a melhorar a usabilidade e a experiência do usuário.

Seção de Tipografia e Iconografia

Tipografia

A tipografia é um elemento essencial do design system, ela que define a hierarquia visual e a legibilidade da nossa interface. No MyBuddy, utilizamos a fonte "Inter", uma fonte sans-serif moderna e legivel, perfeita para interfaces digitais. A fonte é usada em diferentes pesos e tamanhos para criar uma hierarquia visual clara e consistente em toda a interface, seguindo os conceitos do CSS. Além disso, utilizamos a fonte "DM Sans" para destacar elementos importantes, criando um contraste visual interessante e ajudando a guiar o olhar do usuário para as informações mais relevantes.

Iconografia

Os ícones são elementos visuais importantes no nosso sistema de design, eles ajudam a comunicar informações de forma rápida, e melhorar a aparência do nosso produto. No MyBuddy, optamos por utilizar a biblioteca de ícones "Material Icons", por oferecer uma ampla variedade de ícones e possuir uma estilização moderna e consistente, que se encaixa perfeitamente com a identidade visual do nosso produto. Os ícones são usados para representar ações, estados e informações de forma clara e intuitiva, ajudando a melhorar a usabilidade e a experiência do usuário.

Seção de Espaçamento e Componentes

Espaçamentos

Os espaçamentos são muito importantes para um design consistente, eles que definem o espaços entre elementos e ajudam a criar "respiro visual" na interface, e consequentemente melhorando a legibilidade e usabilidade do produto. No MyBuddy, utilizamos um sistema de espaçamento baseado em uma escala modular, onde os espaçamentos são definidos em múltiplos de 4px, como 4px, 8px, 16px, 32px, etc. Isso ajuda a criar uma hierarquia visual clara e consistente em toda a interface, e facilita o desenvolvimento de novos componentes e layouts.

Componentes

Os componentes são os blocos de construção do MyBuddy, eles que definem a estrutura e a funcionalidade da nossa interface. No MyBuddy, definimos uma variedade de componentes, como os campos de formulário, checkboxs e buttons. Esses componentes são projetados para serem reutilizáveis, acessíveis e fáceis de usar, seguindo as diretrizes do design system e utilizando as cores, tipografia e espaçamentos definidos anteriormente. Os componentes são usados para criar uma interface consistente e intuitiva, ajudando a melhorar a usabilidade e a experiência do usuário.
